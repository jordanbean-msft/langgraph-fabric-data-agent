# langgraph-fabric-console

Interactive terminal interface for the LangGraph Fabric Data Agent sample.
Streams agent responses directly to the terminal.

## Modules

| Module | Purpose |
| --- | --- |
| `console.py` | Interactive terminal surface with streaming |
| `main.py` | Entrypoint (`langgraph-fabric-console` script) |

## Run

```bash
uv run langgraph-fabric-console
```

See the [architecture guide](../../docs/architecture.md) for full details.
